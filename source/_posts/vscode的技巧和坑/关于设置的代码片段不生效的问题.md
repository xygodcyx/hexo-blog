# 操作步骤
在 vscode 的 setting.json 里添加需要使用代码片段的语言:

例如:

```json
  "[markdown]": {
    "editor.quickSuggestions": {
      "comments": "on",
      "strings": "on",
      "other": "on"
    }
  },
  "[javascript]": {
    "editor.quickSuggestions": {
      "comments": "on",
      "strings": "on",
      "other": "on"
    }
  },
```
